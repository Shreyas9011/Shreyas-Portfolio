# Shreyas Shirsawade | Personal Portfolio 🚀

A modern, responsive, and highly optimized personal portfolio built to showcase my professional experience, projects, and skills as a React Frontend Developer. Designed with a sleek, permanent dark theme and built-in interactive animations.

![Portfolio Preview](./src/assets/profile.jpg) 

## ✨ Features

* **Sleek Dark Theme:** A permanent, high-contrast dark mode tailored for developers, using tailored Tailwind CSS color palettes.
* **Interactive UI:** Includes a custom scroll progress bar, smooth scrolling navigation, and subtle gradient animations.
* **Fully Responsive:** Mobile-first design that looks great on desktops, tablets, and smartphones.
* **Component-Based:** Clean and maintainable React architecture for easy updates and scalability.
* **Performance Optimized:** Built with functional components and React hooks for fast rendering.

## 🛠️ Tech Stack

* **Frontend Framework:** React.js
* **Styling:** Tailwind CSS
* **Icons:** Inline SVGs
